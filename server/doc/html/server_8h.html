typedef struct command_s command_t

A structure to represent a command.

This structure represents a command. It contains the command string, a function pointer to execute the command, and the number of arguments required for the command.

◆ server_t

typedef struct server_s server_t

A structure to represent a server.

This structure represents a server. It contains a flag to indicate if the server is running, the port number, the socket, a select configuration, lists of pending clients, graphic clients, and AI clients, server options, and the world.
