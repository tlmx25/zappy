◆ option_t

typedef struct option_s option_t

A structure to represent server options.

This structure represents server options. It contains the port number, the width and height of the world, the number of clients, the frequency of the game, and the names of the teams.

◆ option_tab_t

typedef struct option_tab_s option_tab_t

A structure to represent a mapping between option flags and their parsing functions.

This structure represents a mapping between option flags and their parsing functions. It contains a flag string and a function pointer to the corresponding parsing function.
