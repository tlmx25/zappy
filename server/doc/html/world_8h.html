◆ world_t

typedef struct world_s world_t

A structure to represent the world in the game.

This structure represents the world in the game. It contains the number of teams, a pointer to the tiles in the world, a pointer to the teams in the world, a pointer to the list of eggs in the world, and a pointer to the list of incantations in the world.
