typedef struct egg_list_s egg_list_t

A structure to represent a list of eggs.

This structure represents a list of eggs in the game. It contains pointers to the head and tail of the list, and the size of the list.

◆ egg_t

typedef struct egg_s egg_t

A structure to represent an egg in the game.

This structure represents an egg in the game. Each egg has a position, a team name, an id, and pointers to the next and previous eggs in the list.
